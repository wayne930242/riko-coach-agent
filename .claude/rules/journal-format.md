# 健康日誌格式規範

讀取或修改日誌檔案時，遵守以下規範：

## 檔案命名
- 格式：`YYYY-MM-DD.md`
- 每天只能有一個日誌檔案

## 必要章節結構
每個日誌檔案必須有這些元素：
- H1 標題：`# 健康日誌｜YYYY-MM-DD`
- 水平線分隔各章節

## 章節 Emoji 對應（保持一致）
- `## 💭 心情` — mood
- `## 💪 健身` — workout
- `## 🥗 飲食` — food
- `## 😴 睡眠` — sleep
- `## 📝 隨筆` — note
- `## 今日總結` — summary（由 plan-reviewer 產生）

## 記錄格式
每筆記錄前加時間戳記：
```
- HH:MM — 記錄內容
```

## 禁止事項
- 不可刪除使用者已有的記錄
- 不可修改時間戳記
- 不可合併不同日期到同一檔案
